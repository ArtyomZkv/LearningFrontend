День 2: 
Узнал про атрибуты async и defer. Они игнорируются DOM деревом (построение продолжается), а сами 
скрипты работают в фоновом режиме. Разница между async и defer в том, что скрипт с атрибутом defer запускается только после 
построения DOM дерева. Сама же загрузка скрипта работает в момент построения дерева. Иначе говоря, событие DOMContentLoaded 
ждёт загрузки и выполнения скрипта с атрибутом defer.

Прочитал про переменные. Ничего нового не узнал, за исключением разницы между let и var. let обладает областью видимости
только в рамках блоках кода {}. var - в рамках функции или глобально. 

Типы данных в JS: 
String - строка; Для экранизации кавычек внутри значения перемнной используется \. const company = "Бюро \"Рога и копыта\""
Интерполяция в String: "${variable_name}"
Number - число (от -2^53 до 2^53);
BigInt - число за пределами Number; Для определения в конце числа добавляется суффикс n. 
Boolean - true false;
Undefined  - представляет значение undefined, значение не установлено; Используется для создания неинициализированной переменной.
Null - представляет значение null, значение отсутствует;
Symbol - применяется для обращениям к свойствам сложных объектов (?);
Object - комплексный объект. const user={name: "Tom", age:24};

Язык JS - язык с динамической типизацией. Переменные могут менять значения в ходе работы программы.
